pure-lit

A Todo list app written with pure-lit.

To get started with pure-lit, all you have to do is to execute the following command in your project:

      npm install pure-lit

Compare the code you need for a standard Lit-Element with one where pure-lit was used to see the amount of code you can remove:

Before

After

To make it more convinient, you might want to install lit-element-state-decoupler and lit-element-effect as well, which play very nicely with pure-lit and offer you the first hooks you will want.
